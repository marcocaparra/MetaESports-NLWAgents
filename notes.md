# HTML
- Hypertext
    - Textos puros, livros, revistas, jornais
    - Links: acessar outros documentos
    - Imagens, vídeos, áudio
- Markup
    - Linguagem de marcação
    - Tags: <tag> conteúdo </tag>
    - Elementos: tags, atributos, valores
    - Atributos globais: id, class, style
    - Exemplos: <p>, <h1>, <a>, <img>
- Language
    - Linguagem de marcação
    - Estrutura hierárquica
    - Semântica: significado dos elementos
    - Apresentação: CSS
    - Comportamento: JavaScript
_________________________________________________________________________

# HTTP
- Hypertext Transfer Protocol
    - Protocolo de comunicação
    - Cliente-servidor
    - Requisições e respostas
    - Métodos: GET, POST, PUT, DELETE
    - Códigos de status: 200, 404, 500
_________________________________________________________________________

# URL
- Uniform Resource Locator
    - Endereço de um recurso na web
    - Protocolo: http, https, ftp
    - Domínio: www.exemplo.com
    - Caminho: /pasta/arquivo.html
    - Parâmetros: ?id=123&nome=teste
_________________________________________________________________________

# IP
- Internet Protocol
    - Endereço IP: identificação única de um dispositivo na rede
    - IPv4: 32 bits, formato decimal pontuado (ex: 192.168.0.1)
    - IPv6: 128 bits, formato hexadecimal (ex: 2001:0db8:85a3:0000:0000:8a2e:0370:7334)
_________________________________________________________________________

# DNS
- Domain Name System
    - Sistema de nomes de domínio
    - Tradução de nomes de domínio em endereços IP
    - Hierarquia: domínios de nível superior (TLD), domínios de segundo nível, subdomínios
    - Exemplo: www.exemplo.com -> 192.168.0.1
_________________________________________________________________________

# CSS
- Cascading Style Sheets
    - Folhas de estilo em cascata
    - Estilização de documentos HTML
    - Seletores: id, class, tag, pseudo-classes
    - Propriedades: cor, fonte, margem, padding, layout
    - Responsividade: media queries, flexbox, grid
_________________________________________________________________________

# JS
- Linguagem de programação
- Browsers
- Input -> Procces -> Output
- Variáveis
- Functions (Funções)
    - Agrupamento e reuso de códigos
    - Sequência lógica
    - Saída
- Gemini (IA): Esperar uma resposta
- Mexer no meu navegador
- Estutura de decisão
- Estrutura de dados
- Algoritmo: sequência de passos, de maneira ordenada, a fim de chegar a alguma conclusão (limitada)
